# 訂單出貨排定系統

### 情境:

公司接到訂單後，可安排出貨日程及整理商品需求資訊，分析數據了解銷售狀況

### 功能:

- 可新增預定出貨資訊
- 可依地區篩選

### 待加入功能:

- 編輯出貨資訊
- 刪除出貨資訊
- 依照日期排序
- 出貨/未出貨頁籤
- 出貨數據整合......
