# 訂單出貨排定系統

### 情境:

公司接到訂單後，可安排出貨日程及整理商品需求資訊，分析數據了解銷售狀況

### 功能:

- 可新增預定出貨資訊
- 可依地區篩選
- 編輯出貨資訊
- 刪除出貨資訊

### 待加入功能:

- 依照日期排序
- 出貨/未出貨頁籤
- 預定出貨圖表

### 使用工具

- [Validate.js](https://validatejs.org/#validate-js)
- [JSON Server](https://github.com/typicode/json-server)
